## Herramienta de predicción del aislamiento a ruido aéreo mediante distintos métodos

Esta herramienta utiliza cuatro modelos distintos para predecir el índice reducción sonora por tercios de octava para los materiales de una base de datos provista en formato Excel. Los modelos matemáticos utilizados son:

* Modelo de Cremer.
* Modelo según la norma ISO 12354-1.
* Modelo de Sharp.
* Modelo de Davy.

En el sector izquierdo deben colocarse las dimensiones del material a simular (que puede elegirse entre las opciones de la base de datos provista con el selector de material) y luego debe presionarse el botón para aplicar las dimensiones y el material. Una vez hecha esa selección pueden compararse los resultados según cada modelo activándolos según conveniencia con los selectores de modelo inferiores. Si se lo desea, puede guardarse un archivo en formato Excel presionando el botón de guardado. Esto creará una tabla con los datos por tercio de octava junto a un gráfico similar al que puede verse en el visualizador de la herramienta.

Para utilizarlo, solo es necesario colocar todos los archivos en la misma carpeta y luego ejecutar el archivo "TP1.py".
